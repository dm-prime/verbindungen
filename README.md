# nyt connections game but everything in german

## admin stories

- convex as backend for saving the boards
- admin login via convex auth
- way to seed admin account password
- some boards have a day assosciated with them
- past boards are in a random past games pool
- create 10 day boards, 5 in the past, as a sample
- add an admin feature to crud the boards in case the game admin wants to adjust some words

## user stories

- users can login via anonymous or email (first time code)
- in convex, we shall store their game status (played, datetime, won/lost)
- user can check game history. Make sure if a user played a past board after the day of that board, it shows up when it was played, no mistakes there
- user can open the current day board and play it
- user can open a random board from past tool, it displays on which date it was published, and can play it
